# From Zero to Dev Container: A Live Workshop

**Event:** DevOpsDays Amsterdam 2024  
**Date:** March 7, 2024  
**Location:** Amsterdam, Netherlands

## Abstract

Dev Containers are one of the most underutilized tools in the modern developer's toolkit. In this hands-on workshop, we go from zero to a fully configured Dev Container in under 90 minutes.

Attendees set up their own dev containers live, covering Node.js, Python, and .NET project types. We also explore Dev Container Features, dotfiles integration, and how to use the same container definition with GitHub Codespaces.

## What We Build

1. A basic Node.js dev container from scratch
2. Adding Dev Container Features (git, GitHub CLI, Docker-in-Docker)
3. Customizing VS Code extensions and settings
4. Connecting to GitHub Codespaces
