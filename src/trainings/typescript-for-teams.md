# TypeScript for Teams: From Beginner to Production

**Duration:** 3 days (24 hours)  
**Audience:** JavaScript developers, frontend and backend engineers  
**Format:** In-person or virtual (max 16 participants)

## Overview

TypeScript adoption is growing fast, but many teams struggle to move beyond the basics. This workshop takes developers from TypeScript fundamentals through advanced patterns, with a focus on what actually matters in production codebases and for team collaboration.

## Curriculum

### Day 1: Foundations
- TypeScript fundamentals: types, interfaces, enums, generics
- Configuring `tsconfig.json` for your project
- Migrating existing JavaScript incrementally
- **Hands-on:** Migrate a small JavaScript module to TypeScript

### Day 2: Intermediate Patterns
- Advanced types: unions, intersections, mapped types, template literals
- Generics in depth: constraints, defaults, variance
- Error handling patterns: Result types, discriminated unions
- Working with external types: DefinitelyTyped, declaration merging
- **Hands-on:** Build a type-safe API client

### Day 3: Production Patterns & Tooling
- TypeScript at scale: project references, composite projects
- Performance: understanding and improving type-check times
- Testing with TypeScript: jest, vitest, type-level tests
- Linting and formatting: ESLint, prettier, type-aware rules
- CI/CD integration and type safety in pipelines
- **Hands-on:** Set up a production-ready TypeScript monorepo

## Prerequisites

- Comfortable writing JavaScript (ES6+)
- Basic understanding of Node.js
- No TypeScript experience required

## What's Included

- All course materials and exercises
- Take-home reference card: TypeScript patterns cheat sheet
- ESLint config template for TypeScript projects
- Access to workshop repository with all examples

## Outcomes

After this workshop, participants will:
- Write idiomatic, type-safe TypeScript with confidence
- Know when to use advanced type features (and when not to)
- Have a toolkit of patterns for common TypeScript challenges
- Be able to configure and maintain TypeScript tooling
